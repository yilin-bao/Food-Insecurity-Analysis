countyBasic Questionranking\_model module
=============================================

Ranking model description:
---------------------------

This file will implement a model to capture county food insecurity ranking in California.It takes 5 factors into account: overall food insecurity(fi) rate, density of food insecure people, child fi rate, budget shortfall and fi changes in recent 2 years. The result will be visualized by a choropleth map where a dark color corresponds to a severe degree of food insecurity.
